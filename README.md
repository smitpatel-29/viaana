# Viaana Foundation Clone

Project setup for cloning the [Viaana Foundation](https://viaanafoundation.org/) website.
Using HTML, CSS, and Tailwind CSS (via Vite).

## Setup & Run

1. Open a terminal in this directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the displayed URL in your browser.
